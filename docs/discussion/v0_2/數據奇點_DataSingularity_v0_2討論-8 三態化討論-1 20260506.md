### **三、虛數計算是否就是「逆向實數計算」？**

是，但有更深的结构。

逆向實數計算 \= 將正向公式中的符號反轉，用 `-1` 取代 `+1`：

`text`

`正向：ΔΦ = π · |δ_d| · |sin(θ)|    （δ_d = +1）`

`逆向：ΔΦ = π · |-δ_d| · |sin(θ)|   （δ_d = -1，絕對值相同）`

但這只是實軸上的鏡像，ΔΦ 仍然是正實數（因為取絕對值）。真正的虛數計算應該發生在複數平面上，讓 ΔΦ 本身攜帶虛部。

### **四、完整的虛數計算：相位變成複數**

`text`

`ΔΦ 在實軸：來自 δ_d = +1 的驅動（交斥正旋）`

`ΔΦ 在虛軸：來自 δ_d = -1 的驅動（交融逆旋）`

若正向實數計算生成 `Φ_total ∈ ℝ⁺`（螺旋的實軸投影），則虛數計算生成的是 `Φ_total ∈ iℝ`（螺旋的虛軸投影）。

`text`

`Φ_complex = Φ_repel + i·Φ_attract`

`Φ_repel  = Σ ΔΦ 當 δ_d = +1    （交斥累積，實部）`

`Φ_attract = Σ ΔΦ 當 δ_d = -1    （交融累積，虛部）`

`text`

`H_k = H_k⁺ + i·H_k⁻`

`Φ_complex = Φ_repel + i·Φ_attract`

### **一、現有結構中的乘除對偶**

`text`

`n_R` 是實軸上的積累，`n_L` 是虛軸上的積累。若將比例因子視為複數，則：
