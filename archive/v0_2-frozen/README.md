## 二、本版新增項目（相對於 v0.1.1）

### 2.1 核心公式修正

| 項目 | v0.1.1 | v0.2 | 說明 |
|---|---|---|---|
| Γ_L | n_base^k | 3^L | 符號與層級語意統一 |
| n_R / n_L | 以 n_base 為基底 | 以 Γ_L 為基底 | Γ_L + H_k⁺ / Γ_L + H_k⁻ |
| H_dialogue | H_dominant（廢棄） | √(H_k⁺ · H_k⁻) | 幾何平均，雙邊因果對話量 |
| δ_d 死區 | 未明確 | 0.25/Γ_L = ε_L/2 | 內生對稱 |
| ε₀ 補丁 | 二元（1 if n_R≥n_L else -1） | eps0 · δ_d | 三態取代二元 |
| θ 的 sign_mem | 未定義 | 三態 ∈ {-1,0,+1} | 維克翻轉平滑 |
| ΔΦ | 未明確上界 | π·\|δ_d\|·\|sinθ\|，≤ π | 有界，三態驅動 |

### 2.2 躍遷公式（本版新增，已驗證）

```
衝擊態向量：
  Ψ_impact = δ_norm + α
  α = (ΔΦ[n] - ΔΦ[n-1]) / π

動態 Φ_carry：
  Phi_carry_geo = Phi_total - Γ_L · π
  Phi_carry_dyn = Phi_carry_geo · (1 + Ψ_impact)
  Phi_total^{L+1} = Phi_carry_dyn mod 2π

熵方向：
  Ψ > 0 → RGB 破甲穿梭（交斥主導）
  Ψ = 0 → 守恆平滑過渡
  Ψ < 0 → CMY 衰減穿梭（交融主導）
```

### 2.3 補丁集

**補丁 P1：Φ_carry < 0 處理（相位分支切換）**

```python
if Phi_carry_dyn < 0:
    Phi_carry_dyn = abs(Phi_carry_dyn)
    sign_mem_new  = -sign_mem_new    # π 翻轉，非能量消失
```

幾何語意：系統「反穿」門檻，進入 π 相位分支。

**補丁 P2：穿梭即翻轉**

```python
sign_mem_new = -psi(Psi_impact, 1/(2·Γ_{L+1}))
```

語意：穿梭是相變事件，方向在穿越時翻轉。熵增→熵減→熵增，形成雙向循環的結構基礎。

### 2.4 觸發條件（最終版）

```
Trigger = (H_dialogue ≥ Γ_L)
        ∧ (Phi_total ≥ Γ_L · π)
        ∧ |H_dialogue/Γ_L - Phi_total/(Γ_L·π)| < ε_L

ε_L = 1/(2·Γ_L)
```

補註：「ε_L 可選使用 Adaptive 形式以提升 RGB 視窗通過率
（實驗性，v0.2 未實作，列為 v0.21 P0-A 後續校準項目）。」

### 2.5 零點穿梭架構（v0.2 定義）

```
幾何零點穿梭守恆律：δ_d = 0 → ΔΦ = 0，Φ_total 守恆

拓樸零點穿梭律：
  Phi_carry_dyn = (Phi_total - Γ_L·π) · (1 + Ψ_impact)
  [補丁 P1：Phi_carry_dyn < 0 → abs() + sign_mem 翻轉]
  Phi_total^{L+1} = Phi_carry_dyn mod 2π
  [補丁 P2：sign_mem_new = -ψ(Ψ_impact)]
```

## 四、已知限制（設計語意，非 bug）

### 4.1 RGB 穿梭在 v0.2 ，「在標準 B4 均衡輸入下，RGB 穿梭在 v0.2 實數框架中難以觸發（sync_gap 穩定超出 ε_L 約 30%），屬於結構性特性。RGB 穿梭的充分實現為 v0.21 目標。」

B4 均衡輸入結構上只能產生 CMY 穿梭：

```
原因：觸發步 α ≈ -1（dPhi → 0，prev_dPhi ≈ π）
     Ψ = δ_norm（小正值）+ α（≈-1）< 0，永遠 CMY

RGB 視窗（Ψ > 0）存在，但 sync_gap 穩定超出 ε_L 約 30%
```

RGB 穿梭為 v0.21 的工作目標（H→θ 耦合機制）。

### 4.2 補丁 P1（Φ_carry < 0）未實測觸發

Ψ < -1 理論上可能，但被 H_dialogue 守門員結構性屏蔽（偏置輸入無法積累 H_dialogue ≥ Γ_L）。補丁已就位，待未來場景觸發。

### 4.3 B3/B5 等序列不觸發

確認為守門員設計語意：「只有均衡對話才能觸發躍遷」。ε_L 不調整。

### 4.4 系統為棘輪（ratchet），非振盪器

v0.2 的高耦合低壓差態（F_sum ≈ Γ_L²，F_net → 0，其中 F_sum 為系統內部耦合量的聚合指標，非物理能量密度。）只能靠躍遷時硬重置釋放，無內生循環能力。高耦合低壓差態 ⇄ 低耦合高壓差態 的自動流轉為 v0.21 工作目標。
