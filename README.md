# **摘要**

數據奇點（Data Singularity）v0.1 是 ESP v0.1 公設骨架的計算實作層。本文件建立三元三態系統的完整數學結構，包含三個基礎定義、核心守恆等式、動態勢阱、非離散與離散的因果交融交斥關係，以及數據奇點的可操作定義。

核心命題：非離散（張力）與離散（H）通過投影算子 P^{-1,0,1} 因果交融交斥。守恆等式 \-1/n\_L \+ 1/n\_R \= 0 是兩者交融的不變量，是兩者交斥時的失衡指標。t \= \-1/n\_L \+ 1/n\_R 是演化的時間生成子，等效於 0點樞紐的度規曲率 1/n\_0。

# **二、核心命題與守恆等式**

| 核心命題：非離散（張力，連續場）與離散（H，整數）通過投影算子 P^{-1,0,1} 因果交融交斥雙向連動。守恆等式 \-1/n\_L \+ 1/n\_R \= 0 → 交融時：兩者的不變量（n\_L \= n\_R）→ 交斥時：失衡的指標（n\_L ≠ n\_R）。時間生成子 t \= \-1/n\_L \+ 1/n\_R → t 不是獨立的外部參數 → t 由左右邊界的不對稱性本身生成 |
| :---- |

## **2.1　守恆等式的完整表述**

**∀穩定態 k：-1/n\_L(k) \+ 1/n\_R(k) \= 0　即 n\_L \= n\_R**

**∃奇點 k：　-1/n\_L(k) \+ 1/n\_R(k) ≠ 0　→　t ≠ 0　→　躍遷**

**躍遷後：　　-1/n\_L'(k) \+ 1/n\_R'(k) \= 0　守恆重建**

n\_L \= n\_R \= n\_base 的邊界條件（原初態，k=0）：

\-1/3 \+ 1/3 \= 0（守恆成立）

0點退出吸斥計算，作為傳導空間存在

lim(n\_L, n\_R → n\_base)\[-1/n\_L \+ 1/n\_R\] \= 0（極限守恆）

## **2.2　0點樞紐的度規曲率**

定義 0點樞紐算子 n\_0：

$$\\frac{1}{n\_0} \= \-\\frac{1}{n\_L} \+ \\frac{1}{n\_R}$$

**推論：**

$$t \= \\frac{1}{n\_0}$$

即**時間生成子 t 等效於 0點樞紐的度規曲率**。

| n\_0 狀態 | 條件 | 物理意涵 |
| :---: | :---- | :---- |
| **n\_0 → ∞** | n\_L \= n\_R | 曲率趨零，t \= 0，系統守恆，0點無限開放 |
| **n\_0 \> 0** | n\_R \> n\_L | 曲率為正，t \> 0，正旋，交融主導 |
| **n\_0 \< 0** | n\_L \> n\_R | 曲率為負，t \< 0，逆旋，交斥主導 |
| **n\_0 → 0** | 極度失衡 | 曲率極大，奇點逼近，躍遷觸發 |

n\_0 的正負號自動內建拓樸旋向，n\_0 趨近於 0 時即為數據奇點的幾何臨界。**n\_0 的幾何展開留待 v0.2 正式定義。**

## **2.3　三個方向值與奇點強度**

| 守恆等式值 | 物理意涵 | 對應熵方向 |
| :---: | :---- | :---- |
| **\-1/n\_L \+ 1/n\_R \= 0** | 守恆（平衡） | 平衡態，n\_L \= n\_R，原初態或拓樸不動點 |
| **\-1/n\_L \+ 1/n\_R \> 0** | 熱寂（正向失衡） | 熵增方向，n\_R \> n\_L，+1 主導，交融超越交斥 |
| **\-1/n\_L \+ 1/n\_R \< 0** | 退回（負向失衡） | 熵減方向，n\_L \> n\_R，-1 主導，交斥超越交融 |

奇點強度定義：

**奇點強度 \= |-1/n\_L \+ 1/n\_R|**

強度趨零　→　趨向守恆態（n\_L ≈ n\_R）

強度增大　→　躍遷驅動力增強

強度趨近極限值　→　熱寂或退回極限

## **2.4　時間值域**

t \= \-1/n\_L \+ 1/n\_R 的完整值域：

| t 值域 | 狀態 | 說明 |
| :---: | :---- | :---- |
| **t \= 0** | 守恆（平衡） | n\_L \= n\_R，原初態或拓樸不動點 |
| **t \> 0** | 熱寂方向 | n\_R \> n\_L，熵增，正旋，交融主導 |
| **t \< 0** | 退回方向 | n\_L \> n\_R，熵減，逆旋，交斥主導 |

# **三、三個基礎定義**

| 定義一：狀態空間 s ∈ {-1, 0, \+1}，三個離散狀態，先驗存在，不依賴外部定義。對應 ESP v0.1 公設一 |
| :---- |

| 定義二：歷史張量 H\_k \= Σ sᵢ（i \= 1 到 k）。歷史張量是所有輸入狀態的累積和。H\_0 \= 0（原初態，歷史張量=0）。每次輸入推進一個演化步數 k，H 更新一次 |
| :---- |

| 定義三：奇點條件 ∃k：H\_k \= n\_k（累積張量達到層次臨界值），且 ΔH\_k ≈ Δn\_k（張量增量等於層次邊界增量），且 P\_endo ≠ P\_exo 但 H(P\_endo) \= H(P\_exo)（路徑不同但結果相同）。奇點條件的物理意涵：ΔH ≈ Δn 不是張力趨零，而是張力達到當前層次之局部極值（local maximum under n\_k constraint），無法在當前層次釋放。H 停止增長 \= 當前層次容量飽和。H=n 是最小不可再分的單向累積結構。n\_base=3 是第一層基礎臨界值（1+1+1=3）。 |
| :---- |

# **四、非離散與離散的因果交融交斥**

## **4.1　兩個層次的定義**

| 層次 | 性質 | 說明 |
| :---: | :---- | :---- |
| **離散（H）** | 整數，有邊界，可枚舉 | 狀態的位置，歷史的記錄 |
| **非離散（張力）** | 連續，無邊界，不可枚舉 | 狀態之間的流動，演化的動力 |
| **關係** | 因果交融交斥雙向連動 | 兩者互為邊界，互為條件 |

## **4.2　雙向連動機制**

交融方向（非離散→離散）：

張力累積（連續過程）→ H 達到整數值（離散跳躍）

連續的張力流動被 P^{-1,0,1} 量子化為離散步驟

交斥方向（離散→非離散）：

H 的邊界 n\_L, n\_R（離散閥值）→ 約束張力的活動空間

離散結構給連續張力設定邊界條件

## **4.3　投影算子 P^{-1,0,1}**

投影算子定義：

**P^{-1,0,1}(張力) \= (s₋₁, s₀, s₊₁)**

| 分量 | 映射規則 | 說明 |
| :---: | :---- | :---- |
| **s₋₁** | 張力在 \-1 方向的投影 | H 的負向貢獻（交斥分量），累積至 H^-\_k |
| **s₀** | 張力在 0 方向的投影 \= 0 | 守恆，0點退出計算，永遠為0 |
| **s₊₁** | 張力在 \+1 方向的投影 | H 的正向貢獻（交融分量），累積至 H^+\_k |

**H\_k \= H^+\_k \- H^-\_k　（淨張量）**

0成分永遠為0——原初0點守恆在投影算子裡自動成立，不需要額外設定。

## **4.4　奇偶因果交融交斥**

| H 奇偶性 | 非離散（張力）狀態 | 守恆等式 |
| :---: | :---- | :---- |
| **偶數 H（2,4,8...）** | 張力對稱，n\_L ≈ n\_R，內生演化 | \-1/n\_L \+ 1/n\_R \= 0　偶數 H 傾向於張力對稱（statistical tendency），但不構成充分條件 |
| **奇數 H（3,5...）** | 張力不對稱，n\_L ≠ n\_R，失衡 | \-1/n\_L \+ 1/n\_R ≠ 0　躍遷驅動 |
| **H \= n\_k（奇點）** | 張力達到當前層次之局部極值，無法釋放 | 失衡達到當前層級局部極值，強迫躍遷 |

**演化節律之統計特性**  
系統之奇偶交替構成其**主要演化節律（Dominant Pattern）**。奇偶交替呈現演化節律傾向（tendency），但實際是否守恆或躍遷仍由 n\_L / n\_R 決定。須注意此節律為系統之演化統計特徵，而非單次事件之充要條件。

# **五、動態勢阱　n\_k(-n\_L, 0\_k, n\_R)**

## **5.1　層次算子 n\_k 與左右邊界**

**n\_k \= f(n\_L, 0\_k, n\_R)**

左右邊界由歷史張量的方向性累積決定：

$$n\_L \= n\_{base} \+ H^-\_k \\quad \\text{（負向邊界，交斥累積，} n\_L \> 0\\text{）}$$

$$n\_R \= n\_{base} \+ H^+\_k \\quad \\text{（正向邊界，交融累積，} n\_R \> 0\\text{）}$$

其中 **n\_base \= 3**（最小可分基礎結構）

**時間生成子由邊界不對稱性生成：**

$$t \= \-\\frac{1}{n\_L} \+ \\frac{1}{n\_R}$$

- k=0 時：H^-\_0 \= 0，H^+\_0 \= 0，故 n\_L \= n\_R \= 3，t \= \-1/3 \+ 1/3 \= **0**（原初守恆）✅  
- H^+\_k 增大（交融主導）→ n\_R 增大 → t 趨正（熱寂方向）✅  
- H^-\_k 增大（交斥主導）→ n\_L 增大 → t 趨負（退回方向）✅

0\_k　→　當前動態平衡點（帶有歷史方向性記憶）

**三倍累積結構：**

$$3\_0 \= 1\_0 \+ 1\_0 \+ 1\_0$$

n\_base=3 是最小不可再分的基礎臨界值，由三個單位累積而成，是動態勢阱的根節點。

## **5.2　靜態守恆 \= 動態守恆 \= 非穩定平衡守恆**

三者是因果交融交斥的同一守恆性質：

| 守恆表述 | 數學形式 | 本質 |
| :---: | :---- | :---- |
| **靜態守恆** | k=0，n\_L=n\_R=3，t=0 | 非穩定平衡的靜態表現 |
| **動態守恆** | n\_k 持續更新，-1/n\_L+1/n\_R=0 | 非穩定平衡的動態表現（k≥1） |
| **非穩定平衡守恆** | 每個解都是暫時的 | f(n\_L,n\_R,k)持續生成暫時解，螺旋演化本質 |
| **熱寂極限** | n\_R \>\> n\_L，t\>0 正向極值 | 非穩定平衡的正向極限 |
| **退回極限** | n\_L \>\> n\_R，t\<0 負向極值 | 非穩定平衡的負向極限 |
| **回歸原初** | 新的 n\_L'=n\_R'=n\_base'，t=0 | 攜帶精萃的新守恆不動點 |

f(n\_L, n\_R, k) 不是在找穩定解，而是在持續生成非穩定的暫時解——這是螺旋演化的數學本質。

## **5.3　自相似遞歸結構**

動態勢阱具有自相似遞歸性質：

**n\_k(-n\_L(-n\_L, 0\_k, n\_R), 0\_k, n\_R(-n\_L, 0\_k, n\_R))**

外層：n\_k(邊界L, 0\_k, 邊界R)

邊界L \= n\_k(-n\_L, 0\_k, n\_R)　左邊界本身是一個勢阱

邊界R \= n\_k(-n\_L, 0\_k, n\_R)　右邊界本身是一個勢阱

遞歸終止條件：

**t \= \-1/n\_L \+ 1/n\_R \= 0　→　內生終止（原初態，守恆不動點，n\_L \= n\_R）**

**t \= \-1/n\_L \+ 1/n\_R ≠ 0　→　外化終止（拓樸不動點，繼續遞歸）**

內生外化因果交融交斥：n\_k(-3, 0, 3\) 是兩種終止條件的交匯點，也是整個自相似遞歸的根。

## **5.4　三層同構**

**{-1, 0, \+1}　←→　t(-t, 0\_k, \+t)　←→　n\_k(-n\_L, 0\_k, n\_R)**

三個層次完全同構，由因果交融交斥統一驅動。熵的時間箭頭在 0\_k 偏移為 H^±\_k 的瞬間產生（即 n\_L ≠ n\_R 的瞬間）。

# **六、數據奇點定義**

## **6.1　正式定義**

| 數據奇點（Data Singularity）定義與躍遷條件 數據奇點定義為系統演化步數 $k$ 滿足臨界條件 $H\_k \= n\_k^\*$ 之瞬間。 其中 $n\_k^\* \= \\text{scalarize}(n\_k)$ 為邊界結構之純量化截面映射（即 $|n\_k|$）。 當系統處於奇點狀態時，滿足以下特徵： 動態臨界：$\\Delta H\_k \\approx \\Delta n\_k$，即張量增量與層次邊界增量同步。 路徑等效：$P\_{endo} \\neq P\_{exo}$ 但 $H(P\_{endo}) \= H(P\_{exo})$，內生演化與外生突變路徑在局部不可區分。 張力極值：守恆等式暫時失衡（$t \\neq 0$），奇點強度 $|-1/n\_L \+ 1/n\_R|$ 達到當前層次之局部極值（Local Maximum under n\_k constraint）。 不可逆穿越：$H=n\_k$ 為因果對撞之穿越點而非停留點。系統於此完成不可逆之質變躍遷，隨後新層次之邊界重置，對稱守恆 $-1/n\_L' \+ 1/n\_R' \= 0$ 重新成立。 | | :---- |

## **6.2　奇變點正式表述**

每個演化步數 k（k \= 1, 2, 3, ...）都存在奇變點，奇變強度隨 k 以 3^k 指數增長：

| k 演化步數 | 1 | 2 | 3 | 4 | ... | k |
| :---: | :---: | :---: | :---: | :---: | :---: | :---: |
| **奇變強度** | 3^1 | 3^2 | 3^3 | 3^4 | ... | 3^k |
|  | (3) | (9) | (27) | (81) |  |  |

**奇變點的正式表述：**

k \> 0（正旋，熵增方向，t \> 0）： $$\\text{奇變點} \= (-3^k\_0,\\ 0\_k,\\ 3^k\_0)$$

k \< 0（逆旋，熵減方向，t \< 0）： $$\\text{奇變點} \= (-3^{-k}\_0, 0\_{-k}, 3^{-k}\_0)$$

k \= 0（原初態，t \= 0）： $$(0\_0)$$

**奇變點與層級展開基數**  
奇變點同步表述內生與外生層級之躍遷。兩相鄰奇變點間，外生演化表現為受內生驅動的漸進式非線性增長。

* **奇變強度定義**：$\\text{Strength} \= 3^k$。  
* **標度聲明**：$3^k$ 為當前版本採用之標準化奇變強度標度（standardized singularity strength scale），完整推導與動力學來源將於 v0.2 正式定義。  
* **註記**：$3^k$ 為 n\_base=3 的遞歸放大（multiplicative expansion over recursive levels），即三元分支結構在每層遞歸中的展開基數。其動力學推導與嚴格證明留待 v0.2 工作規範進行。

## **6.3　路徑重合量化**

**|P\_in · P\_out| / (‖P\_in‖ × ‖P\_out‖) → 1　完全重合，奇點觸發**

# **九、權重結構**

**吸力 1/3　＋　斥力 1/3　＋　張量場 1/3　＝　1**

| 狀態 | 結構權重 | 計算層對應 | 說明 |
| :---: | :---- | :---- | :---- |
| **\+1** | 1/3 | 斥力 1/3 | 因果交斥來源，s=+1，累積至 H^+\_k |
| **0** | 1/3 | 張量場 1/3（0×1/3=0） | 傳導空間，s=0，無吸斥力，淨張力為零 |
| **\-1** | 1/3 | 吸力 1/3 | 因果交融來源，s=-1，累積至 H^-\_k |
| **張量** | 2/3 | 吸力+斥力 | H\_k 的方向性貢獻 |
| **張量場** | 1/3 | 0點場域 | 傳導空間，高通透性，非能量死寂 |

**結構權重 vs 動態權重：**

- **結構權重（各 1/3）**：系統靜態機率分佈，對應原初態 s ∈ {-1, 0, \+1} 的先驗等比  
- **動態權重（各 1/2）**：0\_k 成為傳導媒介後，退出能量交換，剩餘吸力與斥力在動力學層面對撞，此時各佔 1/2，對應 \-1/n\_L \+ 1/n\_R \= 0（n\_L \= n\_R）的對稱結構  
- **0 的動力學角色**：0 不構成作用力，其在動力層對應為傳導流形（transmission manifold），不參與吸斥對撞，但決定作用的傳遞與重組結構

兩者不矛盾——前者描述存在分配，後者描述作用分配。

**三元基底核心假設：**  
本框架在三元基底上建立結構與動力的對稱基礎。拒絕以二元動力學替代三元幾何，是數據奇點計算結構的核心假設。三元結構（-1, 0, \+1）之完整對齊，構成動態勢阱與度規曲率計算的先驗基礎。

本文件中「force」一詞為廣義結構性作用（structural interaction），不等同於物理學中之力（force）定義，特別是 0 分量不對應任何能量做功概念。   
The term "force" in this document denotes a structural interaction and does not correspond to the physical definition of force. In particular, the 0-component does not perform work.
